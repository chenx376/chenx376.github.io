网球发球技巧

发表于 2018-05-14 | 分类于 技能树 | 0 comments

本篇博客原文转自链接

网球发球五要领

（1）握拍：作为初学者可用东方式正手握拍。大陆式握拍，即半东方式正手和半东方式反手握拍可介绍给有一定水平的初学者。

（2）准备动作/站位：双脚齐肩宽，在端线后侧身站立。右脚与底线基本上平行。左脚正对右网柱。手腕和手臂放松握拍于身体前。左手在拍颈处托住拍。两脚尖的对角线就对着目标。

（3）向后拉拍和抛球：两手臂一同向下和向上运动。球从伸展的左手中向上垂直抛出。位置在身体前面和左脚上部。握拍手掌在向后拉拍时朝下。身体重心平稳地向前脚移动。抛球的高度应能满足击球手臂的充分伸展，并使击球感到舒适。

（4）有力的环绕动作：抛球后，身体开始向前转动，球拍在身后做环绕动作，并最后向前挥动击球。

（5）击球点：运动员必须尽力伸展身体，在最高点击球。击球点应在身体右前方，基本上与右肩充分伸直相一致。击球时手臂和球拍充分伸展。击球时，身体的转动和身体重心的向前转移，以达到右鞋底正对后挡网。理想的要求是，从球拍的顶部到左脚后跟成一直线。

网球发球直角法则

第一个直角：发球时，让你的身体与发球手臂的大臂形成一个直角。

第二个直角：发球手臂的大臂与小臂之间形成另一个直角。

当然,你不能像木头人一样，僵硬地维持直角，而应该在发球过程中的某一个瞬间经历这样一个姿势。尝试90度的发球技巧，可以180度地改善发球。

网球发球抛球技巧

手执球的手法不恰当

不少初学者，手执球的手法是，手掌上翻，掌面向上，拇指和四个手相同时握球，将球上抛。此手法在手臂向上举时，临近抛球的，容易造成四个手指以及手腕轻微地钩的动作。我在这里对此种握法只认为不恰当，因为有些运动员经过反复练习，也习以抛的相对准确，但对初学者就有难度。我建议改变一下手握球的方式，不妨一试。可以采取手事直立，掌心侧向，拇指与食指握住球的两侧，中指托任球的底部，手臂上举抛球在临近出手之际，手腕向上挺起，而不要钩手腕，此种抛球因受人体关节的限制，不易产生的多余动作，对球上抛影响较少。

向上抛球的定义不是很准确，应是向上送球为宜

不少初学者在向上抛球时，一般都采取用前臂用力向上抛球，球出后的高度在肩部左右，球出手与上抛的高点距离较长。球在空中距离越长，出现偏差越大。为什么说是向上送球更为恰当呢？辅助手在挥拍手向后拉拍的同时，手臂上举，以属为受点，当手臂上举伸直时，球出手，利用手臂上学的惯性，轻轻将球送上去，这样球上工后的高度偏差不会太大。三、在后担拉拍转体时，因为抛球手也开始上举，抛球手臂容易随同转体向右转动，使抛球手臂在球出手的，处于转体后的身前，而不是在体侧。这也是初学者容易出现的错误，使球送上去出现误差。这是发球技术掌握的一个难点，可以反复练习转体的同时，两手各自向相反方同举起。

球抛不准，是要通过单独反复练习，有能得到提高

练习方法可以按发球的准备姿势站定，在头的上方选择一个合适的高度，设个目标，如头上方的某一树叶，树枝等物，反复将球按照动作要领，向上送出，反复练习，提高手上感觉，经过一段时间，必然会取得一定进步。然后再进一步，在做抛球的同时，加进转体后拉拍的协同动作，这样就和逐渐克服抛球不准的毛病，使发球得到提高。

网球发球

业网球选手都非常地清楚，通过网球磨砺发球有效改善比赛局面。这也是很多专业网球选手最常使用一种发球战术法。网球比赛离不开发球，与其他技术相比，它是惟一可以完全由球员自己控制的击球。对正向中级水平过渡的人来说，掌握了正确的发球技巧和运用策略，往往能极大改善比赛的状况。我们不得不承认，发球是你球技大长之后需要突破的最后的一道关口了。

初级

目标：抛球和挥拍动作协调流畅。

刚学发球时，用不着多深奥的理论，你只要站到底线上的发球区域，集中精力让抛球和挥拍动作协调流畅地配合起来就行了。至于什么发球线路、腿部或上身的运动，你都不用管。

持拍练习空挥，效果其实并不好。建议你暂时放下球拍，让持拍手握住一个有点儿质量的物体（如果有可能，小一点儿的橄榄球比较合适），再练习抛球和挥拍。这样会帮助你专心体会手臂的协作。等到自己能够放松、准确地完成挥拍动作后，再拿起球拍练习，效果就很容易出来了。

若是想练习用上双腿的话，可以像平时发球一样摆好准备姿势，前脚（也就是非持拍手一侧的脚）跨出一步，然后练习抛球和投掷橄榄球。这一练习的好处就是能促使你在发球中习惯充分扭转肩膀——这恰恰是发好球的必要因素。

再来谈谈抛球的问题。非持拍手臂向上抛球，然后伸直手臂，在击球动作中再慢慢放下，听起来并不困难。但能让两只手臂运动的速度和节奏相互协调起来却并不容易。通常情况下，球被抛出后，伸直的抛球手臂没有多少运动的空间了，因而挥动的速度应当慢一些。

否则，抛球就要很高，才能让球拍有足够的时间挥到击球区。当然，如果抛球手臂运动得太慢，或者持拍手臂运动得太快，为了赶在合适时间进入击球区，球拍将不得不在挥动中停顿一下。让两只手臂协调运动、互不干扰永远是发好球的不二法则。

目标：让发球成为制造先机的武器，全面运用旋转和力量给对手的接球制造麻烦。

初学者的发球更像是个开球仪式，但对处于中级水平的人来说，发球则是掌握主动的第一步。发球的线路在这阶段里变得很重要，球要尽可能落在对手最难接的地方，比如发向对手的身体和内外两个角落。另外，你也可以通过改变自己的站位来控制不同的角度。

图1：向上擦过球身，会帮助你发出上旋球，这对打出稳定而有效的二发很有好处。

挥拍放松、流畅、自如，就能提高发球的速度。试试这种方法：非持拍手不要真的抛出球，双臂做完整的击球动作；随挥结束的时候，持拍手臂经由一个圆弧的轨迹，重新回到引拍的位置，重新开始无球的空挥；等第二次随挥结束后，你就可以将球抛出，做出一次真正的发球动作。这一练习对运用肩膀和加速拍头很有好处。

旋转对中级球员来说，是提高发球安全性的好办法。旋转可以让你在充分挥拍的情况下仍能保证将球发进有效区域内，而对手接起来却很困难，因此在二发中尤其有用。要想让自己的发球产生上旋效果，你不仅需要弯曲膝盖，还要让拍面向上擦过球身。抛球刚好进入身体靠前的上方，就能容易地帮你找到擦出上旋的感觉了。

高级

目标：充分的肩膀转动和膝盖弯曲，让一二发都令对手头痛。

处在高级阶段，能用肩膀是不够的，能将肩膀转到合适的程度才是重要的。抛球的时候，对手能看到你的大部分后背吗？前球王桑普拉斯是个夸张的例子，他发球时转肩得相当剧烈，以至于后背都能和球网平行了。而对普通人来说，能将肩膀扭转到和球网呈30度到40度的程度就很不错了。

当选择两脚一前一后开立的发球站姿时，你不必让前脚与底线平行，只要呈45度角就行了，脚尖冲向网柱。然后当你抛球的时候，弯曲膝盖、顶出髋关节时，双脚就会自然提起，变得与底线平行。这样的准备姿势能先将身体充分侧向发球的方向，为发球时上肢和肩膀的扭转提供方便。

膝盖弯曲得越好，双腿的运用就越自然，来自脚蹬地的作用力转移到击球中的力量也就越大。在你打定了进攻的主意之后，发球时就会自然向前移动重心，身体被自然而然地带离地面，进入球场。你可以自己观察一下，结束发球后如果你的身体没有进入底线以内的话（至少有一只脚跨入底线），那就是你还没有充分用到双腿。有了肩膀充分的转动，膝盖的弯曲，你就能将球发到发球区的外角，把对手带到球场之外了。

高水平球员在发球中常犯的问题就是挥拍太过剧烈，因为紧张导致手臂僵硬。有一种方法可以减轻这种问题：发球时只用拇指和食指钩住球拍。这样做能有效地防止握拍紧张、放松手掌肌肉，反而能更好地为拍头加速。

网球的发球技术，是网球比赛第一局开始的关键。所以我们不能在第一个发球就输掉，当你拿到第一局的发球权，那就会让自己有更多的赢取机会。当然，如果你处于下风时，更需要通过网球磨砺发球有效改善比赛局面了，这是打网球的一种战术法。其实在一些网球发达的国家里，他们的教练都是要求掌握战术后再练习技术。这就是打网球的美妙之处。
