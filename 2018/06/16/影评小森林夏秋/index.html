小森林-夏秋 影评

发表于 2018-06-16 | 分类于 书影摘 | 0 comments

在生活中，我们大多时候过分注重结果，而忽略过程的美好。

简介

本部电影主要讲的是女孩市子，由于无法适应都市快节奏的步伐，最终回到乡下的老宅，并过着日出而作日落而息的生活，用身边的食材做出一道道朴素而美味的菜肴。

电影中，百分之90的语音，都是来自市子的内心独白。但是观众在看整部电影的过程，却丝毫不会感到乏味；一方面是因为整个电影优美的乡村风光，另一方面来源于对一道道菜肴的期待。

总的来说，是一部让人看完，觉得非常舒服的治愈系作品。在欣赏电影的过程中，观众只需要沉浸在电影所创造的氛围中，感受的市子的用心生活，以及小森林的美好。

平凡的精致

电影中拍摄的每一道菜，其实仔细想想，都不算什么“大菜”。除湿器的同时，利用火炉顺便烤面包；用大米和酵母做成的酸米酒；胡颓子做成的果酱；以及自家特制的伍斯特酱油。

但是，每一道菜，都浸入了市子的耐心和认真；因而每一道菜看起来都如此的诱人。

为了做胡颓子果酱，需要到树下坚持捡起一颗颗掉落的胡颓子；之后将它们一一捣烂；最恼人的是要把胡颓子的籽一粒粒的捡出来；然后搅拌导入锅中做成果酱。想想，真是需要很大的耐心才能完成。在做完这一系列过程之后，终于把没有人愿意吃的酸涩的胡颓子，变成美味的果酱。抹在面包上，放入嘴中，酸甜的果酱配上面包，相信一定不错。

自家酿的酸甜的冰镇米酒，在炎热的夏天来上一杯，实在太适合不过了;利用自家天地里生长的西红柿，做出的盐水西红柿罐头，搭配意面，让人嘴馋；用山上捡到的板栗，剥皮放入有小苏打的水中，然后加入白糖或其他调味料慢慢煮，最后变成了甜香软糯的栗子，只能吞几下口水以表敬意了。

另一道让人印象深刻的菜变成炒青菜，其实炒青菜实在再简单不过了。不过要做出像市子妈妈做的爽脆的感觉，便要费一番功夫，需要将每个青菜的筋，一一剥去，实在是很费功夫；但是，正是因为有这一番别样的功夫，青菜才能变得更加爽口美味

所以市子真是一个努力认真生活的女孩，一个人需要劈柴，割稻草，收获这一年的粮食和蔬菜，然后以备不时之需。有时甚至去帮忙搬鱼赚取外快，到山上收获一些野果，核桃、板栗、胡颓子，做成一道道能够装点生活的蔬菜

最简单的一道菜应该是烤面包吧，在湿热的梅雨季节，为了不浪费炉火中的余温，于是做一个烤面包；从砍柴开始，到点火，反复揉面。忙活了那么久，最后也仅仅做成一个面包？是啊，即使是一块简单的面包，也不是轻而易举就能得到的，浸入了反复的辛劳，配上自己的果酱，真的做到物尽其用了

在物质匮乏的农村，如果想过上精致的生活，就需要反复的用心；在夏天的时候去采集胡颓子和榛果，做成不同的果酱；在秋天，采集核桃和板栗，混杂成可口的小吃。这样精致的生活，需要对每一样食材都用心对待，这样才能将平凡的它们，化腐朽为神奇的，变为舌尖上的美味

市子向大城市的我们展示了，即使是最简单的食物，不管是果酱，面包甚至是酱油，都是需要很多的步骤才能制作而成的。看完这些，更让人感激科学带来的成就，我们可以以每场便宜的价格买到各种各样的食材，江河湖海的鱼虾，深山里的蘑菇，香脆的杏仁腰果榛子，这些其实都需要费很大功夫才能得到呢!

所以，之后吃东西的时候，好好享受每一样食材吧；即使是最简单的食材，他们都凝聚了很多人的劳动。如果有机会自己做饭的话，用心对待每一样食材，这样自己吃起来的时候，心里也会感受到温暖呢！（笑）
