CFA计算器使用

发表于 2018-04-25 | 分类于 技能树 | 0 comments

这部分主要整理一下CFA计算器相关的内容，CFA能够使用两种计算器：

  1. 德州仪器的BA II Plus(or Professional)
  2. 惠普的12C(or Platinum)

这篇文章仅对第一种计算器做介绍，需要注意的是，BA计算器与大家能够熟练使用的Casio计算机还是有一定不同。如果真的想备战CFA，还是需要认真的尝试着按几次。

自己的笔记

与卡西欧计算器类似的是，德州的计算器也支持公式运算。但是不支持公式显示，所以运算的时候还是按照正常的公式运算按键，具体的公式，需要自己存储在脑子中。

  1. 第二排的2ND按钮是启动第二级运算用的

  2. 如何暂时储存：算出的结果可以使用STO + 数字，暂时存储起来，方便下一次使用。还需要使用的时候，用RCL + 数字调出; 这个时候直接Clear Work 并不会消去在暂存中的数据。这个时候，也可以通过 2ND + 0，查看memory 中的数字。可以通过↑和↓进行查看，进入这种模式下，再按clear work,所有数据会被清零

  3. 如何计算时间间隔：2ND + 1可以进入计算Date的漠视，例如7-14-2005的输入方法是7.1405。注意：2005的20是默认的（什么愚蠢的设定，服了）。另外需要注意的地方是，数字数据不是按=号，而是按ENTER,第一排第二个键，然后利用↓进行日期输入。计算也是按CPT，第一排第一个键进行结果计算

  4. 如何使用ICONV模式进行EFF(Effictive Annual Rate)计算，2ND + 2进入此模式，NOM 是Normal Rate, C/Y 是每年计息次数，之后与之前时间的是一样的。C/Y进入之后，还可以按照之前普通计算的方法，例如24/5等的计算方法进行计算

  5. 如何计算Time Value of Money:

    • N : Number of Years or number or payments
    • I/Y : Interest rate per period
    • PV : Present Value
    • FV : Future Value
    • PMT : Payments(for annuities)

注意：这个地方对计算器的使用与之前不一样，对于计算器第三排的按钮。都是先按数字，然后再按按钮进行赋值，比如N = 4，那么就先按4再按N。

注意：有End model和begin model表示每期结算发生在结束还是开始，两种情况必须分开来看。如果想要切换，先2ND + PMT(第三排倒数第二个) + 2ND + Enter进行模式切换

注意：FV 与 PMT，PV的正负号关系，至今还没搞懂什么时候正，什么时候负

一级常用的三种方法

此部分内容转自高顿网校

货币时间价值5要素

货币时间价值解决了不同时间点的等价关系，现在的钱在时间的作用下，未来会有不同的价值，而未来的到手的钱也可以通过一定的折现率得到一个现在的货币价值。

那么我们在计算器计算时就会涉及到相关的货币时间价值5要素哦，有哪五要素呢？掏出你的德州计算器来，第三行的那一排白色按键 就是货币时间价值5要素。

  • N 是现值到终值一共隔了多少期
  • I/Y 每期利率
  • PV 现值就是最开始0时刻对应的现金流或者说价值
  • PMT 每期期间现金流
  • FV 终值
  • 时间轴可以帮助你很好的理解货币时间价值5要素，如图:

来，看一道例题：An investor will receive an annuity of $4000 a year for ten years. The payment is to be received at the end of the first year. At an annual interest rate of 9%, what is this annuity’s worth at the end of ten years? 这就是一道普通年金的题目，每年年末支付?4000，付10年，年利率9%，问10年后终值是多少？

计算器：

  1. [10] [N]
  2. [9] I/Y
  3. [0] [PV]
  4. [4000] [PMT]
  5. [CPT] [FV] = - 60771.718

NPV&IRR

这两个是在资本项目预算的时候会用到的两个评估方法。

根据时间轴给出的条件求NPV（净现值）

好，做一个项目期初要投一笔钱，这里期初时投了175， 后面4期每期得到不等额的回报，然后呢我们求净现值，就要考虑货币的时间价值，所以会需要一个折现率＝10%，求NPV。

计算器：

  1. [CF] [2ND] [CE/C]
  2. [175] [+/-] [ENTER]
  3. [↓] [25] [ENTER]
  4. [↓][↓][100] [ENTER]
  5. [↓][↓] [75] [ENTER]
  6. [↓][↓][50] [ENTER]
  7. [NPV] [10] [ENTER]
  8. [↓][CPT] =20.87

　　那么其实在这里，我们除了可以用计算器的第二行这个NPV求期间不等额现金流的现值，只要让CF0=0，然后后面算法一样哦。
　　
求IRR

　　IRR是使得净现值NPV=0时刻的折现率，那么计算器的用法上现金流输入方法相同，最后求IRR,直接IRR CPT即可。

　计算器： 　

  1. [CF] [2ND] [CE/C]
  2. [175] [+/-] [ENTER
  3. [↓][25] [ENTER]
  4. [↓][↓][100] [ENTER]
  5. [↓][↓] [75] [ENTER]
  6. [↓][↓][50] [ENTER]
  7. [IRR] [CPT] = 15.067

排列组合

　　这个很简单就是告诉大家排列和组合在计算器的哪里怎么按，这样就可以不记公式。

　　比如吧你有五只股票，卖三只，问对于这三只股票有多少种组合方法。

　　[5] [2ND][+] [3] = 10 （组合符号在加号上面哦 按2ND就可以把它调出来）

　　还是上面的题干，稍微改一下，股票买卖的先后顺序是有关系的，那么问依次卖掉三只股票有多少种方法。

　　[5] [2ND] [–] [3] = 60 （同理，排列在减号上面！）
