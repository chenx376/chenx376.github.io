土味情话集

发表于 2018-07-03 | 分类于 杂谈 | 0 comments

嘿嘿嘿嘿嘿嘿，这篇主要是收集了一些土味情话

  • “请管好你的嘴。”

    “为什么？”

    “因为我随时会亲它。”

  • “你可以笑一个吗？”

    “为什么啊？”

    “因为我的咖啡忘加糖了。”

  • “你想喝点什么？”

    “我想呵护你。”

  • “你知道情人眼里出什么吗？”

    “西施啊。”

    “不，出现你。”

  • “你觉得你适合穿什么衣服？”

    “好看的衣服？”

    “不，你适合被我征服。”

  • 最近外面有谣言说我喜欢你

    我要澄清一下

    那不是谣言。﻿﻿

  • “你最近是不是又胖了？”

    “没有啊，为什么这么说？”

    “那你为什么在我心里的分量越来越重了呢？”

  • “天气这么冷，你知道怎么取暖吗？”

    “不知道。”

    “我也不知道，但我知道怎么娶你。”

  • “你属什么？”

    “我属虎。”

    “不，你属于我。”

  • 你知道我的心在哪边么？

    左边啊

    不，在你那边

  • 我们来玩木头人不许动吧

    好！我输了

    为什么？

    因为我心动了

  • 我结婚你一定要来

    为什么？

    因为没有新娘会很尴尬

  • 不想撞南墙了，

    只想撞你的胸膛 　　

  • 被你赞过的朋友圈，叫甜甜圈

  • “你有打火机吗？”

    “没有啊。”

    “那你是怎么点燃我的心的？”

  • “你猜我想喝什么？”

    “不知道啊。”

    “我想呵护你。”

  • “你猜我想吃什么？”

    “不知道啊。”

    “痴痴地望着你。”

  • “你累不累啊？”

    “不累。”

    “可是你都在我心里跑了一天了。”

  • 你今天怪怪的。

    怎么了？

    怪好看的

  • 我觉得你好像一款游戏。

    什么游戏？

    我的世界。

  • 我在找一匹马。

    什么马？

    你的微信号码。

  • 你是哪里人？

    湖南人。

    不，是我的心上人

  • 情话是我抄的，但只说给你听是真的。

  • 我觉得我最近可能有点怕你

    为什么？

    我怕老婆

  • 你知道如果你喜欢一个人的话，应该怎么去表达？

    如果是我的话，我就会一直看着她

  • “最近我看到你就觉得头很晕”

    “为什么？”

    “因为爱情使人头昏脑胀”

  • “我觉得你这个人越看越无趣阿”

    “为什么”

    “让人越看越想睡”

  • “我觉得你的嘴唇有点问题哦”

    “什么问题？”

    “离我有点远”

  • “我问你，如果你是一款大衣，你认为你是哪款大衣？”

    “长款大衣”

    “错！你是我心里的唯一”

  • “木头做的门叫什么门？”

    “木门”

    “铁做的门叫什么门？”

    “铁门”

    “那幸福做的门叫什么门？”

    “？？？”

    “ 我 们 ”

  • “可以借我一床被子吗？”

    “为什么？”

    “这样，我就欠你一辈子了”

  • “我跟你玩捉迷藏好不好？”

    “好哇”

    “不行哎我肯定会被你发现啦”

    “哈？”

    “因为喜欢一个人，藏都藏不住”
