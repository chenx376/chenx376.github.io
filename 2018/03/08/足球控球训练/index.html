足球控球训练

发表于 2018-03-08 | 分类于 技能树 | 0 comments

本篇文章转自虎扑, 主要是包含了一些基本的控球训练(主要是颠球)

颠球基础

正确部位触球

脚背是最重要的触球部位，用正确的部位练习可以保证你日后颠球的稳定性。

我们所说的脚背就是你鞋带这个位置，也就是图中黄点的部位。

首先，在这里我们要强调练习时你要感受脚背将球弹起的感觉，这点非常重要，只有这样你才会不断的尝试用正确的部位去触球。我们开始用你惯用脚的脚背将球弹起。

记住用正确的部位触球，即脚背的中间位置触球。

将球轻轻将其弹起后用手接住，不断重复，然后再用弱势脚的脚背触球；刚开始可能感觉有点难，但是双脚必须都要会，这一点很重要。

上面这些练习的过程就是要找到你正确触球的感觉，也就是让你找到那个“熟悉的味道”。

当我们很熟悉这个感觉后，可以尝试下用双脚的脚背交替弹球接住；

让球落下，用惯用脚脚背将球弹起，别用手接球让它再次落下，再用弱势脚的脚背将其弹起，然后再用手将球接住；

如果此时你能不接球连续颠球，那你基本已经会颠球了

但是，颠球并没有那么简单，更多的小伙伴都是像下面这样，顿时失去了身体协调性，依然是磕磕绊绊，连续颠3个都是很难的事。

为什么呢？虽然你可以通过第一个练习能让自己确保足球可以直上直下的颠起，但是此时你还没有建立你颠球的节奏，击球的时机不对会导致你身体重心会轻易丢失，让你疲于应付。

单脚反弹颠球练习

所以别着急，下面才是今天的关键，对初学者来说，我们让球从地上弹起后再颠球，这样有助于你建立颠球的节奏。

从地面弹起后颠球很重要，你明显可以感到这比不反弹颠球简单多了。

刚开始练习落地反弹颠球只用你的惯用脚，当球从地面弹起落下后，用惯用脚的脚背将其弹起，再次让球从地面弹起用脚背颠起；

在这个过程中你要确保你用脚背中间触球，如果触球部位过于靠上；

那么球可能会蹦的很远不容易控制，相反地如果触球部位过低，会让你无法及时进行第二次的连续触球

在这个练习的过程中，你要确保球颠起的高度，同时为了跟上球的移动，你可以改变自己站位的姿势。用这种落地反弹的方法练习颠球，直到你可以持续颠10次。

双脚反弹颠起练习

接下来我们练习落地反弹后双脚颠球，让球从地面弹起，落下后用惯用脚脚背颠起，再让球自由落下，用弱势脚脚背将球颠起

这种双脚交替颠球，很容易让你找到颠球的节奏，会提高你控球时的自信，持续练习直到你可以双脚交替连续不断颠10次

单脚二次反弹颠起练习

如果你是一名初学者不要过早跳跃到不反弹颠球，循序渐进很重要，从反弹颠球一点点过渡到直接颠球。

现在我们练习地面反弹用惯用脚连续不落地颠球二次，然后再让球落到地面弹起再用脚连续颠两次。

每一次都用脚背中间位置颠球，让球可以在空中直上直下，第二下要比第一下尽量柔一点，让他变得有点节奏。

双脚交替反弹颠球练习

然后再过渡到双脚开弓交替颠球，让球落下，首先用惯用脚接球将球颠起，待球落下后用弱势脚将球再颠起，两个脚分别触球一次，再让球落地反弹进行下一轮颠球。

控球训练

习惯对我们有着绝大的影响，因为它是一贯的，在不知不觉中，经年累月地影响着我们的行为，决定着我们的成功。你此时的能力，不正是多年努力积累的结果吗？

根据行为心理学的研究结果：3周以上的重复会形成习惯；3个月以上的重复会形成稳定的习惯，即同一个动作，重复3周就会变成习惯性动作，形成稳定的习惯。

亚里士多德说：“人的行为总是一再重复。因此，卓越不是单一的举动，而是习惯。”所以，在实现成功的过程中，除了要不断激发自己的成功欲望，要有信心、有热情、有意志、有毅力等之外，还应该搭上习惯这一成功的快车，实现自己的目标。
