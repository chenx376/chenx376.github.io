第六题 539. Minimum Time Difference

题目描述

给定一组24小时制的时间，格式为“小时：分钟”，求任意两组时间中分钟数间隔的最小值。

注意：

  1. 给定时间至少2个，至多20000个。
  2. 给定时间是合法的，范围在00:00到23:59之间。

算法

利用bucket sort进行排序，然后找出最近的两个时间，时间复杂度O(n)
