第六题 215. Kth Largest Element in an Array

题目描述

从一个未经排序的数组中找出第k大的元素。注意是排序之后的第k大，而非第k个不重复的元素

测试样例如题目描述

可以假设k一定是有效的， 1 ≤ k ≤ 数组长度

算法

每次获得队列尾的数字，然后将所有比队列尾数字小的调整到其左边，其余在右边。这样，每次我们就可以知道第k个大的数字，通过这样的方法，可以在不进行排序的情况下，最终获得第k个最大的数字。
