第五题 372. Super Pow

题目描述

计算a ^ b mod 1337的值。a是正整数，b是一个极大的正整数，以数组形式给出。

算法

ab % k = (a%k)(b%k)%k
a^1234567 % k = (a^1234560 % k) * (a^7 % k) % k 
= (a^123456 % k)^10 % k * (a^7 % k) % k
我们可以写出方程
f(a,1234567) = f(a, 1234560) * f(a, 7) % k = f(f(a, 123456),10) * f(a,7)%k;

通过回溯的方法，实现这个方法
